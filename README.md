# The-Education-Tutor-for-Remote-India
Personalized AI tutors are revolutionizing education, but they are expensive to run. In rural India, where internet is spotty and computing power is low, students cannot afford high-latency, high-cost queries to massive models like GPT-4 for every question.

Build an intelligent tutoring system capable of ingesting entire state-board textbooks and providing personalized, curriculum-aligned answers. The system must optimize for lowest cost per query and minimal data transfer.

Ingest large PDFs (textbooks) but answer questions without re-processing the entire document every time.

Demonstrate a significant reduction in API costs compared to a baseline RAG system.
